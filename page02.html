>>Jenis pembolehubah

Di python, ada beberapa pembolehubah.
Antaranya ialah :

  1. String (perkataan biasa / teks)
  2. Interger (Nombor bulat tanpa perpuluhan)
  3. Float (Nombor dengan titik perpuluhan)
  4. Boolean (nilai True dan False sahaja ada)
perkataan = "saya rajin"
umur = 13
harga = 2.3
kawin = True

Di atas ialah 4 jenis-jenis pembolehubah di python, semua berbeza.


perkataan = "saya rajin"

pembolehubah ubah perkataan berisi data jenis string (perkataan)

String mesti ditulis dalam pemetik kata ganda (""), supaya dapat membezakan string dengan interger(nombor)


nombor1 = "10"
nombor2 = 10

keduanya adalah berbeza dari segi jenis data
